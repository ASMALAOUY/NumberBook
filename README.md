##  Description

NumberBook est une application mobile Android complète de gestion de contacts qui permet de :
-  Lire et afficher les contacts du téléphone
-  Rechercher des contacts (locale et distante)
-  Synchroniser les contacts avec un serveur distant
-  Ajouter manuellement des contacts
-  Interface moderne avec Material Design

##  Fonctionnalités

| Fonctionnalité | Description | Statut |
|----------------|-------------|--------|
|  Chargement des contacts | Lecture des contacts depuis le carnet d'adresses |  |
|  Recherche locale | Filtrage instantané des contacts |  |
|  Recherche API | Recherche sur le serveur distant |  |
